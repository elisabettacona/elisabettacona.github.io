Oakley Website

UX & UI Design

Redesign Product Description Page, Navigation and Search were the main focus of the project for the Oakley website, part of the Luxottica Group.
We improved the User Experience aspect working also on the Homepage, Product Listing Page and Technology page, where the brand explain its product's technology. Here a prototype is available if you want to take a look at it.

In the PDP we created a fixed panel for the main information about the product on the right side for desktop that will go on top on the iPad version.
In the tab under the main product other important information are displayed. Scrolling the page description about the technology used on the product will be provided with interactive content to engage the user.

In the PLP we introduced tags, in order to give users an overview about each product, prevent pogosticking and to improve search function. Tags are explained with icons' help and will be related to sports, gender, customazability, new arrivals/bestsellers/other.

On navigation we change the labels in a clearer way in order to be self-explatory. In the mobile version we added icons, on the sport section, to enable users to learn the meaning before entering other pages.

Technology page in our design becomes "Our Lenses", and has a vertical navigation bar on the left side with labels that convey the benefit of each technology and we integrated products within the page.

For the homepage we did two proposal. The first one is more seasonal or for special collection, designed to be used in short period. The second one is classic but reinforce trust with the use of social medias and user generated content.
